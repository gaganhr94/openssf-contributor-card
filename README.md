# OpenSSF Contributor Card

A site that generates personalized "contributor cards" for people who've shipped code to
[OpenSSF projects](https://openssf.org/projects/). Live at
[gaganhr94.github.io/openssf-contributor-card](https://gaganhr94.github.io/openssf-contributor-card/).

> Not an official OpenSSF site. See [`CONTRIBUTORS_OPTOUT.md`](CONTRIBUTORS_OPTOUT.md)
> if you'd like your card removed.

> [!WARNING]
> Early stage of development. The command usage, configuration file formats and
> deployed URL may change.
